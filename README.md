# RawMacro Releases

This repository hosts release binaries for **RawMacro** — a low-latency Windows keyboard macro tool.

The application checks this repo on startup and offers in-app updates through Settings → 업데이트 확인.

## Download
Get the latest version from the [Releases](https://github.com/sosimei/RawMacro-releases/releases/latest) page.
